Description : 
The Number Guessing Game is a fun and interactive command-line game where the player tries to guess a randomly chosen number between 1 and 100. The game provides feedback on whether the guess is correct or not. Players have a limited number of attempts to guess the number correctly.

Features :
Randomly selects a number between 1 and 100.
Allows the user to input guesses.
Limits the number of attempts to 3 or 10 depending on difficulty (Hard or Easy).
Displays the correct number if the player runs out of attempts.
Option to play again after each round.

Technologies Used:
Python
Random module (for number generation)
